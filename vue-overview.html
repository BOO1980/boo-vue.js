Vue Overview

What is vue

  • Vue is a JavaScript framework for building user interfaces.
  • Vue builds on top of standard HTML, CSS and javascript and provides a declarive and component-based programming model that helps you efficently develop user interfaces be they simple or complex

Single-File Components (SF)

  • An SFC is a reusable self-contained block of code that encapulates HTML, CSS and JavaScript that belong together in a .vue file
  • The core feature of Vue is decarative rendering
  • We us a template syntax that extends HTML, we can describe how the HTML should look like base on JavaScript state
  • When the state changes the HTML updates automatically
  • State that can trigger updates when changed are considered reactive.
  • In Vue the reactice state is held in components
  • We can decare reactive state using the data components option
  • Which should be a function that returns an object
  • The message property will be made available in the template
  • This is how we render dynamic text based on the value of the message using the mustaches syntax
  • The content inside the moustaches is not limted to just identifiers or paths we can also use valid javascript expressions
